Layout

It took me some time deciding what layout to use for this website. I wanted something that was simple, clean, and easy to navigate, yet also offered flexibility for future growth. After considering several options, I eventually settled on a 1-column layout, which provides a straightforward, focused experience for the user. This layout puts the content in the forefront, so visitors can quickly digest the information without distraction.
The 1-column layout format is also optimal for mobile making it easier to implement for mobile support. Unfortunately, for this website, mobile screens are not supported as it's not required by the assignment.
